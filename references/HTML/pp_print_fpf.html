pp_print_fpf : Format.formatter -> ('a, 'b) func -> unit

SYNOPSIS
Print a finite partial function to a formatter.

DESCRIPTION
This prints a finite partial function but only as a trivial string `', to the given formatter. Installed automatically at the top level and probably not useful for most users.

FAILURE CONDITIONS
Never fails.

COMMENTS
The usual case where the formatter is the standard output is print_fpf.
